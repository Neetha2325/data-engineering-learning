# Data Engineering Pipeline Project

This project demonstrates an end-to-end data pipeline using Python, Snowflake, and Airflow.

## Steps:
1. Data ingestion using Python (pandas)
2. Data cleaning and transformation
3. Loading data into Snowflake
4. SQL validation and aggregation
5. Workflow orchestration using Airflow

## Tech Stack:
- Python
- Snowflake
- SQL
- Apache Airflow

## Files:
- python_script.py
- sales_data.csv
- cleaned_sales.csv
- aggregation.sql
- validation.sql
- sales_pipeline_dag.py
